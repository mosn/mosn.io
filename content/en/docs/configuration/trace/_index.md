This topic describes trace configurations of MOSN.

`trace` describes the following basic global parameters of MOSN.

```josn
{
  "tracing": {
    "enable": true,
    "driver": "",
    "config": {
      
    }
  }
}
```

## enable

Boolean. Enable or disable trace.

## driver

Currently, supports `SOFATracer` and `SkyWalking`.
